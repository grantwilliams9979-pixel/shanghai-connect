# Best Way To Work Together

You do not need to give me direct account access, and you should not share passwords, 2FA codes, or API keys in chat.

## Best way to work together is this:

1. You keep account control
- You log into GoDaddy and Vercel yourself.
- I give you exact click-by-click actions and exact DNS values to paste.

2. You share non-sensitive details only
- Domain name
- Vercel project URL
- The DNS records Vercel shows (A, CNAME, TXT values)
- Any error messages/screenshots (without personal billing info)

3. I guide each step live
- Add project to Vercel
- Add custom domain in Vercel
- Add DNS records in GoDaddy
- Verify SSL and redirect www/root correctly

## Fastest setup plan

1. Create/sign in to Vercel and import your GitHub repo.
2. In Vercel project settings, open Domains and add your domain.
3. Copy the DNS records Vercel gives you.
4. In GoDaddy DNS, paste those records exactly.
5. Tell me what status Vercel shows (Pending, Invalid, Verified), and I will troubleshoot immediately.

## What to send to begin

1. Your domain name
2. Whether you want primary domain to be root (example.com) or www (www.example.com)
